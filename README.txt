A responsive WordPress theme for the Unlocking New York project with modern navigation and footer design.

== Description ==

Unlocking New York is a modern, responsive WordPress theme designed for the Unlocking New York project. It features a clean design with a mobile-first approach, custom navigation menus, and integrated social media links.

= Features =

* Responsive design that works on all devices
* Custom navigation menus (primary and footer)
* Custom logo support
* Featured image support
* Social media integration
* Widget-ready sidebar
* Translation ready
* SEO optimized
* Fast loading
* Cross-browser compatible

= Theme Support =

* Custom menus

* Post thumbnails
* HTML5 markup
* Custom background
* Custom header
* Editor styles
* Block editor support
* Wide alignment
* Responsive embeds

== Installation ==

1. Upload the theme folder to `/wp-content/themes/`
2. Activate the theme through the 'Appearance' menu in WordPress
3. Go to Appearance > Menus to set up your navigation
4. Go to Appearance > Customize to configure social media links
5. Go to Appearance > Widgets to add sidebar content

== Frequently Asked Questions ==

= Does this theme support custom menus? =

Yes, the theme supports two menu locations: Primary Menu and Footer Menu.

= Can I customize the social media links? =

Yes, you can add your social media URLs in Appearance > Customize > Social Media Links.

= Is the theme mobile responsive? =

Yes, the theme is fully responsive and works great on all devices.
